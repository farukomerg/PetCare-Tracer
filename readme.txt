Proje Konusu:
PetCare-Tracer - Evcil hayvan sahiplerinin hayvanlarina ait saglik, asi, ilac, beslenme, randevu, hatirlatma ve aktivite verilerini tek merkezden yonetebildigi kapsamli dijital takip platformu.

Projede Kullanilan Teknolojiler:
- Java 17
- Spring Boot
- Spring JDBC
- PostgreSQL
- MongoDB
- BCrypt
- JavaFX
- Android Studio (Java)
- Docker Compose
- Prometheus
- Grafana
- k6
- JUnit / Mockito

Klasor Icerigi:
- backend/petcare-backend
  Spring Boot backend kodlari
- admin-panel/petcare-admin
  JavaFX admin panel kodlari
- mobil-app/PetCareMobile
  Android istemci kodlari
- db
  Veritabani schema ve seed scriptleri
- monitoring
  Prometheus ve Grafana konfigrasyonlari
- tests/k6
  Performans test scriptleri
- docs
  Kurulum, kullanim ve ekran goruntusu rehberleri
